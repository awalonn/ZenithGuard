What's New in ZenithGuard

v2.15.0

Version 2.15.0 - Architectural Refactor

Code Quality & Performance

This is an "under-the-hood" update. We've refactored the Data Breach Warning feature to be more efficient. The logic is now centralized in the background service, reducing the amount of code injected into every page you visit. This improves performance and makes the extension more robust and easier to maintain.

Version 2.13.0 - Proactive Security

New: Data Breach Warnings

You will now see a clear, non-intrusive warning at the top of any site you visit that has a known history of major data breaches. This helps you make informed decisions about your data and password security. This feature can be toggled in General Settings.

Version 2.12.0 - The Refinement Update

Code & Performance Refinements

This update focuses on under-the-hood improvements. We've fixed a memory leak related to network logging, consolidated CSS animations to reduce file size, and performed a general code cleanup for a more stable and efficient extension.

Version 2.11.0 - iFrame Hardening & Smart Consent

New: Sandboxed iFrame & AI Cookie Consent

ZenithGuard now automatically applies security restrictions to third-party iFrames to prevent malicious ads. The AI now also actively looks for the most privacy-friendly option on cookie banners and automatically clicks it for you.

Version 2.8.0 - The Unblockable Update

New: AI Anti-Adblock Wall Defeater

Tired of sites demanding you disable your ad blocker? The **"Defeat Wall"** tool in the popup uses AI to find and remove those annoying overlays, giving you back control. We've also redesigned the popup footer into a clean grid to make all our powerful tools easier to access.
