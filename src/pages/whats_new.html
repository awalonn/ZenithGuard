Version 1.1.8 - The AI Leap & Pure Speed

We've skipped straight to the future! ZenithGuard now uses **Gemini 3.0** for all AI tasks. This means smarter "Defeat Wall" fixes, faster response times, and near-perfect element identification.

Massive Filter List Optimization

Our new **Rule Bundling Engine** reduces filter list memory usage by 5,000%. We saved over **50,000 rule slots**, meaning you can add massive custom lists without ever hitting Chrome's "budget" errors.

Persistent AI Fixes

Once ZenithGuard defeats an adblock wall, it **keeps it defeated**. Selectors are now saved in your secure cloud storage and re-applied automatically whenever you visit that site again.

Architectural Refactor

We've completely rebuilt the core content script engine. It is now split into professional, clean modules (`YouTubeProtector`, `SecurityGuard`, etc.), making the extension faster, more stable, and easier to maintain. No more "spaghetti code"!

Version 1.1.0 - The Hybrid Engine & YouTube Fixes

Hybrid Ad Blocking

We've upgraded to a **Hybrid Engine**. We now combine:
1. **Static Network Rules**: 70,000+ extremely fast rules from EasyList/EasyPrivacy.
2. **Dynamic Cosmetic Rules**: Fetched regularly to hide annoying elements that network rules miss.

YouTube "Nuclear" Protection

We fixed the issue where the "Skip Ad" button was hidden. Now:
- The button is **forced visible** so you can always click it.
- A new **"Nuclear Skipper"** bypasses the UI entirely by fast-forwarding ads if they manage to load.
- Ad interception now starts earlier (`document_start`) to block ads before they even request data.

Isolation Mode Restored

The "Isolation Mode" toggle is back in the popup! Quickly isolate the current site to block all third-party requests for maximum privacy.
